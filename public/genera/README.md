# Reference images

Drop one photo per genus here, named after its `id` in `src/data/genera.ts`:

    cumulus.jpg, stratus.jpg, stratocumulus.jpg, cumulonimbus.jpg,
    nimbostratus.jpg, altocumulus.jpg, altostratus.jpg, cirrus.jpg,
    cirrocumulus.jpg, cirrostratus.jpg, contrail.jpg

These appear in the Clouddex grid and detail sheets once a genus is caught.
Use your own photos or CC-licensed/public-domain sky images. They're optional —
the UI falls back to a gradient tile when an image is missing.
